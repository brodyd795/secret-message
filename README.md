# Secret Message App

## No longer maintained or live

This was a tool designed to allow users to send secret messages to others in a
secure manner.

View a video walkthrough [here](https://www.loom.com/share/583aa40bacec4257ab2e2cf8596f4885).

- Next.js
- Tailwind CSS
- MySQL
- Framer Motion
